DFS 生成所有结果

Posted at — Dec 1, 2019

生成所有结果的题，第一时间要想到使用 DFS 和递归来解决。

主要分为以下几步，

  1. 确定 DFS 的跳出条件，对应树结构的子节点。
  2. 确定剪枝条件，这个是最麻烦的，需要对题目进行分析
  3. 确定继续搜索的方向

Leetcode 上的例题有，22，39，40

22. Generate Parentheses

这一题是给出一个数 n，生成所有的括号组合。

因为是所有的括号组合，想到用 DFS 和递归来解决。

首先确定 DFS 的跳出条件，我们可以设置两个变量 left，right 分别维护左右括号的剩余数量， 当两者都为0的时候跳出。

接着确定剪枝条件，其实跳出条件也可以用整个字符串的长度来维护，但是体现不出括号的特殊性，所以设置两个变量 left，right，因为字符串是从左往右增长的，当 right < left 的时候，意味着有右括号先于左括号出来，需要进行剪枝。

最后确定继续搜索的方向，其实就两种选择，要么加左括号，要么加右括号，注意要回溯，如果 out 的值被更改的话。

class Solution {
public:
    vector<string> generateParenthesis(int n) {
        int left = n, right = n;
        vector<string> result;
        string out = "";
        DFS(left, right, out, result);
        return result;
    }
    
    void DFS(int left, int right, string& out, vector<string>& result) {
        if (left == 0 && right == 0) {result.push_back(out); return;}
        if (left > right) return;
        if (left > 0) {
            out += '(';
            DFS(left - 1, right, out, result);
            out.pop_back();
        }
        if (right > 0) {
            out += ')';
            DFS(left, right - 1, out, result);
            out.pop_back();
        }
    }
};

39. Combination Sum

这一题是给一个 target 找到和为 target 的所有组合，想到用 DFS 和递归来解决。

首先确定 DFS 的跳出条件，自然是 target 为0的时候跳出，因为数都是正的，target 为0表示找到了这样的组合。

然后确定剪枝条件，当 target < 0 的时候，无法生成正确结果，需要剪掉。

最后确定搜索方向，和上一题的一个一个字符生成不同，只有两种选择，对于这一题，下一个数字相当于有从当前到最后一个数所有种选择，所以需要用一个 for 循环做遍历，而上一题可以看作是一个小型遍历，只有两种选择。

class Solution {
public:
    vector<vector<int>> combinationSum(vector<int>& candidates, int target) {
        vector<vector<int>> result;
        vector<int> out;
        DFS(0, target, out, result, candidates);
        return result;
    }
    
    void DFS(int cur, int target, vector<int>& out, vector<vector<int>>& result, vector<int>& candidates) {
        if (target == 0) {result.push_back(out); return;}
        if (target < 0) return;
        for (int i = cur; i < candidates.size(); i++) {
            out.push_back(candidates[i]);
            DFS(i, target - candidates[i], out, result, candidates);
            out.pop_back();
        }
    }
};

40. Combination Sum II

和39题很像，区别是39中每个元素可以用无限次，40中每个元素只能用一次，39的数组中没有重复元素，40中有重复元素。

依然一个一个来看，首先是跳出条件，仍然是 target 为0的时候跳出。

然后是剪枝条件，同样是 target 为负的时候剪枝。

修改的主要是搜索方向，第一个需要注意的是，数组需要先进行一次排序，然后因为有重复元素，比如排序后是 1 1 2，会出现两次{1, 2}，需要加一个判断，保证只出现一次，想象成是树的节点，比较好理解，然后是每个元素只能用一次，需要把 DFS 里面的 i 改为 i + 1.

class Solution {
public:
    vector<vector<int>> combinationSum2(vector<int>& candidates, int target) {
        vector<vector<int>> result;
        vector<int> out;
        sort(candidates.begin(), candidates.end());
        DFS(0, target, out, result, candidates);
        return result;
    }
    
    void DFS(int cur, int target, vector<int>& out, vector<vector<int>>& result, vector<int>& candidates) {
        if (target == 0) {result.push_back(out); return;}
        if (target < 0) return;
        for (int i = cur; i < candidates.size(); i++) {
            if (i > cur && candidates[i] == candidates[i - 1]) continue;
            out.push_back(candidates[i]);
            DFS(i + 1, target - candidates[i], out, result, candidates);
            out.pop_back();
        }
    }
};
